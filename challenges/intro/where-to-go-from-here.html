Basic Linux

This option will introduce you to the Linux Operating System and the unix-based terminal. This is necessary for nearly all of the other options, so it is highly recommended That everyone who doesn't have experience with Linux does this option.

What you will learn:
  • How to set up a Linux machine
  • How to navigate the Linux Operating System
  • How to use the Linux terminal
What do you need to know for this option?
  • Nothing! This option is beginner-friendly

Basic Programming

This option will introduce you to computer science and give your resources to continue learning.

  • What resources there are to learn programming for free
